## 環境構築

- プロジェクトのクローン
```bash
# http
git clone https://github.com/markinveng/react-tech-blog.git
# ssh
git clone git@github.com:markinveng/react-tech-blog.git
```

- localhostを建てる
```bash
# runコマンド
npm run dev
```
npm run dev

localhostを建てた後 [http://localhost:3000](http://localhost:3000) にアクセスすることで表示できます。
